Svelte UI is a collection of high-quality, accessible UI components designed specifically for **Svelte 5**. Built from the ground up to leverage the power of **Runes** and **Snippets**, it provides a seamless development experience with a focus on performance, accessibility, and modern aesthetics.

## Why Svelte UI?

The Svelte ecosystem is evolving rapidly with the release of Svelte 5. While there are many component libraries available, few are built natively for the new reactivity system. Svelte UI fills this gap by providing components that feel like first-class citizens in a Svelte 5 codebase.

## Tech Stack

- **Framework:** [Svelte 5](https://svelte.dev/)
- **Headless UI:** [Bits UI](https://bits-ui.com/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/) & [Tailwind Variants](https://www.tailwind-variants.org/)
- **Icons:** [Lucide Svelte](https://lucide.dev/guide/svelte/introduction)
