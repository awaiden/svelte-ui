import { Button, Field, Form, Input } from '@adn-ui/svelte';
import { Aside, TabItem, Tabs } from '@astrojs/starlight/components';

The `Form` component is a simple wrapper for grouping form fields together with consistent spacing and layout.

## Usage

Use the `Form` component to wrap multiple `Field` or form control components.

<Tabs>
  <TabItem label="Svelte">
```svelte
<script>
  import { Button, Field, Form, Input } from "@adn-ui/svelte";
</script>

<Form onsubmit={(e) => { e.preventDefault(); console.log('Form submitted'); }}>
  <Field.Root>
    <Field.Label>Email</Field.Label>
    <Input type="email" placeholder="Email" />
  </Field.Root>

<Field.Root>
	<Field.Label>Password</Field.Label>
	<Input type="password" placeholder="Password" />
</Field.Root>

  <Button type="submit">Log In</Button>
</Form>
```
  </TabItem>
</Tabs>

## API Reference

### Props

| Prop    | Type     | Default     | Description                                  |
| :------ | :------- | :---------- | :------------------------------------------- |
| `class` | `string` | `undefined` | Additional CSS classes to apply to the form. |

## Accessibility

The `Form` component renders as a `<div>` by default but can be used alongside standard `<form>` attributes. Ensure that you use appropriate labels and ARIA attributes within your form fields for maximum accessibility.
