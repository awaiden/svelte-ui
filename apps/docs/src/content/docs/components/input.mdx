import { Input } from '@adn-ui/svelte';
import { TabItem, Tabs } from '@astrojs/starlight/components';

The `Input` component allows users to enter and edit text. It is a wrapper around the standard `<input>` element with themed styling.

## Usage

<Tabs>
  <TabItem label="Svelte">
```svelte
<script>
  import { Input } from "@adn-ui/svelte";
</script>

<Input type="text" placeholder="Enter your name" />
```
  </TabItem>
</Tabs>

## API Reference

The `Input` component accepts all standard `<input>` attributes.

| Prop    | Type     | Default     | Description                      |
| :------ | :------- | :---------- | :------------------------------- |
| `class` | `string` | `undefined` | Additional CSS classes to apply. |

## Accessibility

- The component uses the native `<input>` element.
- Ensure you associate a `Label` with each `Input` for proper accessibility.
