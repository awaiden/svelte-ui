import { Link } from '@adn-ui/svelte';
import { TabItem, Tabs } from '@astrojs/starlight/components';

The `Link` component is a styled anchor element used for navigation or linking to external resources.

## Usage

<Tabs>
  <TabItem label="Svelte">
```svelte
<script>
  import { Link } from "@adn-ui/svelte";
</script>

<Link href="https://svelte.dev" target="_blank" rel="noopener noreferrer">
  Svelte Documentation
</Link>
```
  </TabItem>
</Tabs>

## API Reference

The `Link` component accepts all standard `<a>` attributes.

| Prop    | Type     | Default     | Description                      |
| :------ | :------- | :---------- | :------------------------------- |
| `class` | `string` | `undefined` | Additional CSS classes to apply. |

## Accessibility

- The component uses the native `<a>` tag for proper link behavior and accessibility.
- Ensure that links are descriptive and clear about their destination.
