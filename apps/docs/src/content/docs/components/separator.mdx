The `Separator` component is used to visually divide sections of content. It can be horizontal or vertical.

## Usage

<Tabs>
  <TabItem label="Svelte">
```svelte
<script>
  import { Separator } from "@adn-ui/svelte";
</script>

<div>
  <p>First Section</p>
  <Separator />
  <p>Second Section</p>
</div>
```
  </TabItem>
</Tabs>

## API Reference

The `Separator` component is built on top of [Bits UI Separator](https://bits-ui.com/docs/components/separator).

| Prop          | Type                         | Default        | Description                       |
| :------------ | :--------------------------- | :------------- | :-------------------------------- |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | The orientation of the separator. |
| `class`       | `string`                     | `undefined`    | Additional classes to apply.      |

## Accessibility

- The component uses the appropriate ARIA attributes for a separator.
- If it's purely decorative, it should have the correct role.
