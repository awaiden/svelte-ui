The `Checkbox` component is used to select one or more options from a set. It is built on top of [Bits UI](https://bits-ui.com/docs/components/checkbox).

## API Reference

### Checkbox

| Prop       | Type                         | Default     | Description                                           |
| :--------- | :--------------------------- | :---------- | :---------------------------------------------------- |
| `checked`  | `boolean \| "indeterminate"` | `false`     | The checked state of the checkbox.                    |
| `disabled` | `boolean`                    | `false`     | Whether the checkbox is disabled.                     |
| `required` | `boolean`                    | `false`     | Whether the checkbox is required for form submission. |
| `name`     | `string`                     | `undefined` | The name of the checkbox for form submission.         |
| `value`    | `string`                     | `"on"`      | The value of the checkbox for form submission.        |

## Accessibility

The `Checkbox` component follows the [WAI-ARIA Checkbox Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/checkbox/):

- **Keyboard Support:** Supports `Space` to toggle the checked state.
- **ARIA Attributes:** Automatically manages `aria-checked`, `aria-disabled`, and `aria-required`.
- **Focus Management:** Visual focus indicators are provided.
