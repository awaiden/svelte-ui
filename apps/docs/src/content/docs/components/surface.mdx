import { Surface } from '@adn-ui/svelte';
import { TabItem, Tabs } from '@astrojs/starlight/components';

The `Surface` component is a basic container for content, providing consistent background and padding. It is often used as a building block for more complex layouts.

## Usage

<Tabs>
  <TabItem label="Svelte">
```svelte
<script>
  import { Surface } from "@adn-ui/svelte";
</script>

<Surface class="p-6 border">
  <p>Content inside the surface.</p>
</Surface>
```
  </TabItem>
</Tabs>

## API Reference

| Prop    | Type     | Default     | Description                                   |
| :------ | :------- | :---------- | :-------------------------------------------- |
| `class` | `string` | `undefined` | Additional classes to apply to the container. |

## Accessibility

- The `Surface` component is a generic `div` and does not have specific accessibility features beyond what is provided by the content it contains.
- Use semantic HTML tags if necessary by passing them through the `class` or using a more specific component.
