import { Input, Label } from '@adn-ui/svelte';
import { TabItem, Tabs } from '@astrojs/starlight/components';

The `Label` component provides a text description for a form element, increasing accessibility and providing a target for clicking to focus the associated input.

## Usage

<Tabs>
  <TabItem label="Svelte">
```svelte
<script>
  import { Label, Input } from "@adn-ui/svelte";
</script>

<div class="flex flex-col gap-2">
  <Label for="name">Your Name</Label>
  <Input id="name" placeholder="John Doe" />
</div>
```
  </TabItem>
</Tabs>

## API Reference

The `Label` component accepts all standard `<label>` attributes.

| Prop    | Type     | Default     | Description                      |
| :------ | :------- | :---------- | :------------------------------- |
| `class` | `string` | `undefined` | Additional CSS classes to apply. |

## Accessibility

- The component uses the native `<label>` element.
- It provides a larger click target for associated inputs.
