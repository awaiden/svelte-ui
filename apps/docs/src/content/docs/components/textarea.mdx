The `Textarea` component allows users to enter and edit multiple lines of text.

## Usage

<Tabs>
  <TabItem label="Svelte">
```svelte
<script>
  import { Textarea, Label } from "@adn-ui/svelte";
</script>

<div class="flex flex-col gap-2">
  <Label for="bio">Biography</Label>
  <Textarea id="bio" placeholder="Tell us about yourself..." />
</div>
```
  </TabItem>
</Tabs>

## API Reference

The `Textarea` component accepts all standard `<textarea>` attributes.

| Prop    | Type     | Default     | Description                      |
| :------ | :------- | :---------- | :------------------------------- |
| `class` | `string` | `undefined` | Additional CSS classes to apply. |

## Accessibility

- The component uses the native `<textarea>` element.
- Ensure you associate a `Label` with each `Textarea` for proper accessibility.
